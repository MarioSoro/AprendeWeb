RSSOwl
Es un lector de RSS bastante completo que está disponible en diferentes idiomas, incluyendo el español. Se trata de una aplicación gratuita que se encuentra disponible en Windows, Mac y Linux. Permite recoger, organizar, visualizar y conservar la información de nuestras suscripciones mediante una interfaz gráfica de uso muy simple. Las noticias se pueden exportar para leerlas offline (sin necesidad de estar conectados).

Los manuales para su uso se encuentran en las siguientes webs (para Windows / para Linux -Liferea-). A continuacion os pongo algunos enlaces interesantes sobre las herramientas FeedReader y Liferea:

Ventajas:
  • Permite ordenar por diferentes filtros incluidos los autores.
  • Compartir elementos de noticias en Facebook o Twitter.
  • Selecciona si desea cargar sólo una vista previa de la noticia, o la historia completa.
  • Disponible para Windows.
  • Importar y alimentar directamente de su cuenta de Google, o desde un archivo OPML o XML.

Desventaja:
  • Al importar los feeds, la estructura de alimentación (carpetas) cambian el cual te obliga a volver a revisarlo.
